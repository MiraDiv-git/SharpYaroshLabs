## Виконав: ст. гр. ПД-21 Терьохін М. А.

> [!NOTE]
> Ресторан "Хтивий Краб" є експериментальним рестораном на виніс. Ця система була натхнена макдональдсом, тобто з логікою, що ти робиш замовлення, чекаєш табло з номером свого замовлення та йдеш до каси його забрати, тому тут столиків нема за ненадібністю
